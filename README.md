# PumpSkope

PumpSkope is a real time Solana and Pump.fun token analysis platform designed to track developer wallet activity, token transactions, holder movements, and risk signals.

## Features

Real time dev wallet tracking

 Pump.fun token monitoring
Holder analysis

Risk signal detection

Solana transaction monitoring

Live token analytics

Wallet activity tracking

Responsive UI

Recursive wallet analysis
Branch wallet detection

Wallet to wallet transaction tracking after token launch

Detection of suspicious transfers from developer wallets

 Identification of possible hidden developer-controlled wallets

## Tech Stack

      Node.js
      JavaScript
      HTML/CSS
      Solana Web3.js
      Supabase
      Bitquery
      Helius API

## Project Goal

The goal of PumpSkope is to improve transparency in the Solana meme token ecosystem by helping users monitor suspicious wallet behavior, track token activity, and identify potential risk signals in real time.
