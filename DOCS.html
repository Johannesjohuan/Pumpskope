Recursive Wallet Detection

We detect secondary wallets that receive tokens directly from the developer. If a wallet receives >0.9% of total supply, it is flagged as Recursive.

Common Patterns:

  • Insider allocations
  • Hidden distribution before trading
  • Multi-wallet selling strategies

Holder Analysis

We scan the top holders for suspicious patterns like Top 10 Concentration. This helps you see if a small group controls the market via linked clusters. Each holder shows real-time buy/sell activity with timestamps.
